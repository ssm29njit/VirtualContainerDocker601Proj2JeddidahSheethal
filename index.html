Containerization Using Docker

The idea of optimizing application build for multiple OS has been long floated. With the invention of Virtual Boxes, one can host an Operating System on the hypervisor and have 'mini' machine to run all the apps dependent on that OS. With containerization hardware platforms are not specifically allocated, it is shared among multiple apps running on the same machine using cloud.
